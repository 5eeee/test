# Studio <4:30/> - Telegram Mini App

## Обзор проекта

Studio <4:30/> - это полнофункциональное Telegram Mini App, разработанное для предоставления услуг по разработке программного обеспечения. Приложение включает в себя каталог услуг, калькулятор проектов, портфолио, корзину с оплатой и административную панель.

## Особенности

- **Интеграция с Telegram Mini Apps**: Полная интеграция с платформой Mini Apps
- **Каталог услуг**: Фильтруемые и сортируемые услуги с детальными модальными окнами
- **Корзина покупок**: Полный процесс оформления заказа с интеграцией YooKassa
- **Калькулятор проектов**: Многоступенчатая форма для оценки индивидуальных проектов
- **Портфолио**: Галерея выполненных проектов
- **Админ-панель**: Панель управления для управления услугами, заказами и аналитикой
- **Адаптивный дизайн**: Работает как на мобильных устройствах, так и на настольных компьютерах
- **Темный/светлый режим**: Поддержка темного и светлого режимов с эффектами гласморфизма

## Технологии

- React/TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- @twa-dev/sdk (Telegram Web Apps SDK)

## Запуск локально

```bash
cd studio-mini-app
npm install
npm run dev
```

## Сборка для продакшена

```bash
cd studio-mini-app
npm run build
```

## Развертывание на GitHub Pages

Проект настроен для автоматического развертывания на GitHub Pages через GitHub Actions:

1. Запушьте код в ветку main
2. GitHub Actions автоматически соберет и развернет приложение
3. Сайт будет доступен по адресу `https://<ваш-username>.github.io/<название-репозитория>`

## Интеграция с YooKassa

Фронтенд подготовлен для интеграции с YooKassa. Для полной реализации потребуется бэкенд для подключения к API YooKassa.

## Настройка Telegram Bot

После развертывания:

1. В BotFather используйте команду `/setwebapp`
2. Выберите вашего бота
3. Введите URL: `https://<ваш-username>.github.io/<название-репозитория>/`
